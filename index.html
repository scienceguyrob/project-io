About This Series

project-io is a structured, self-paced introduction to machine learning using Python. It is written for those who are new to programming and builds methodically from Python fundamentals through to advanced evaluation techniques. Every concept is introduced with clear explanations and worked examples before being applied in code.

The series is designed so that each notebook builds on the previous one. Notebooks 0–2 establish Python and programming foundations. Notebooks 3–5 introduce the core data science libraries and classical statistical learning. Notebooks 6–8 cover the main families of supervised and unsupervised algorithms. Notebooks 9–12 focus on evaluation, metrics, and practical model assessment. Notebook 13 covers ensemble learning in depth, from bagging and bootstrap aggregating through to boosting and AdaBoost. The three bonus notebooks provide additional depth on neural networks, imbalanced learning, and mixture models for those who want to go further.

All interactive widgets, visualisations, and datasets are included. No internet connection is required once the container is running. The full source is available at github.com/scienceguyrob/project-io.

The Notebooks

NotebookNotebook 0: A Python Primer for Machine Learning
📁 Notebook_0/Notebook_0.ipynb
A complete introduction to Python for people with no prior programming experience. Covers variables, types, control flow, functions, scope, and the data structures you will use throughout the rest of the series. Finishes with a signpost to NumPy, pandas, matplotlib, and scikit-learn.
NotebookNotebook 1: Introduction to Python Programming
📁 Notebook_1/Notebook_1.ipynb
Introduces JupyterLab and builds core Python skills in the context of a machine learning problem. Covers arithmetic, strings, lists, dictionaries, sets, and tuples before introducing straight lines as the simplest possible decision boundaries and the idea of residual error.
NotebookNotebook 2: Control Flow, Functions, and Decision Boundaries
📁 Notebook_2/Notebook_2.ipynb
Deepens Python fluency with comparison and logical operators, conditional statements, loops, list comprehensions, and custom functions. Develops the decision boundary theme by searching for the best threshold, extending to 2-D boundaries, and introducing the concept of a loss landscape.
NotebookNotebook 3: NumPy, Statistics, and Feature Analysis
📁 Notebook_3/Notebook_3.ipynb
Introduces NumPy as the numerical foundation of Python machine learning: array creation, indexing, broadcasting, and universal functions. Covers data distributions, correlation, Cohen's d, and feature selection, building from descriptive statistics to a complete classification pipeline.
NotebookNotebook 4: Pandas, Optimisation, and the Curse of Dimensionality
📁 Notebook_4/Notebook_4.ipynb
Covers the pandas library in depth — Series, DataFrames, filtering, GroupBy, apply, and file I/O — before connecting tabular data to ML by framing models as parameterised mapping functions. Introduces gradient descent, the loss landscape, and the curse of dimensionality.
NotebookNotebook 5: Regression and Classification
📁 Notebook_5/Notebook_5.ipynb
Builds simple and multiple linear regression from first principles using least squares, then transitions to logistic regression via the sigmoid function, log-odds, and maximum likelihood estimation. Introduces the scikit-learn fit/predict pattern and key regression evaluation metrics.
NotebookNotebook 6: Naïve Bayes, Decision Trees, Random Forests, and SVMs
📁 Notebook_6/Notebook_6.ipynb
Surveys four classical classifiers side by side. Covers Bayes' theorem and Gaussian Naïve Bayes, information gain and the CART algorithm for decision trees, ensemble methods and bootstrap sampling for random forests, and the maximum-margin hyperplane and kernel trick for SVMs.
BonusNotebook 6 Bonus: Introduction to Neural Networks
📁 Notebook_6_Bonus/Notebook_6_Bonus.ipynb
An extended bonus notebook tracing artificial neural networks from their biological origins through the McCulloch–Pitts neuron and the perceptron to the multilayer perceptron. Implements XOR from scratch, introduces activation functions and backpropagation, and closes with a look at why deep learning exists.
NotebookNotebook 7: Visualisation for Machine Learning and Feature Selection
📁 Notebook_7/Notebook_7.ipynb
A practical guide to data visualisation for ML practitioners using matplotlib and seaborn. Covers histograms, KDE plots, box plots, violin plots, scatter plots, correlation heatmaps, and pairplots, then applies them to feature selection using filter, wrapper, and embedded methods.
BonusNotebook 7 Bonus: Imbalanced Learning
📁 Notebook_7_Bonus/Notebook_7_Bonus.ipynb
A focused bonus notebook on the class imbalance problem. Explains why accuracy is misleading on skewed data and introduces data-level remedies — random undersampling, oversampling, stratification, SMOTE, and Borderline-SMOTE — alongside algorithm-level approaches including cost-sensitive learning.
NotebookNotebook 8: Unsupervised Learning and Clustering
📁 Notebook_8/Notebook_8.ipynb
Introduces unsupervised learning through the lens of similarity and distance metrics, then covers four clustering algorithms: k-nearest neighbours, k-means (from scratch and via scikit-learn), hierarchical clustering with dendrograms, and DBSCAN for density-based discovery.
BonusNotebook 8 Bonus: Mixture Models and the EM Algorithm
📁 Notebook_8_Bonus/Notebook_8_Bonus.ipynb
A bonus notebook introducing Gaussian mixture models as a probabilistic alternative to k-means. Derives the Expectation–Maximisation algorithm step by step, covers covariance types, model selection via AIC and BIC, and shows how a fitted GMM can serve as a density estimator for anomaly detection.
NotebookNotebook 9: OPTICS, Dimensionality Reduction, and PCA
📁 Notebook_9/Notebook_9.ipynb
Extends clustering with OPTICS for varying-density data, then pivots to dimensionality reduction. Builds PCA from first principles — variance, covariance matrices, eigenvectors, eigenvalues — before using scikit-learn's implementation for scree plots, biplots, and downstream classification.
NotebookNotebook 10: Classifier Evaluation
📁 Notebook_10/Notebook_10.ipynb
Addresses the subtleties of evaluating classifiers fairly: the memorisation problem, the i.i.d. assumption, and the No Free Lunch theorem. Covers train/test splitting strategies, stratified and weighted sampling, k-fold cross-validation, and the confusion matrix for binary and multi-class problems.
NotebookNotebook 11: Evaluation Metrics and the Bias–Variance Trade-off
📁 Notebook_11/Notebook_11.ipynb
Defines and compares the full suite of classification metrics — precision, recall, F1, specificity, G-mean, balanced accuracy, Cohen's kappa, and the Matthews correlation coefficient — and explains when each is appropriate. Connects model complexity to the bias–variance trade-off and introduces regularisation.
NotebookNotebook 12: ROC Curves, Precision–Recall Curves, and Clustering Evaluation
📁 Notebook_12/Notebook_12.ipynb
Covers threshold-independent evaluation for binary and multi-class classifiers via ROC curves and AUC, contrasts these with precision–recall curves on imbalanced data, and closes with internal clustering evaluation metrics: WCSS, silhouette score, Davies–Bouldin index, and the Dunn index.
NotebookNotebook 13: Ensemble Learning
📁 Notebook_13/Notebook_13.ipynb
A focused study of ensemble methods from first principles. Introduces the wisdom-of-the-crowd intuition before covering bagging and bootstrap aggregating — including out-of-bag evaluation and the connection to Random Forests — then transitions to boosting as sequential error correction. Derives and implements AdaBoost in full, exploring the exponential loss connection, adaptive sample reweighting, the learning rate and n_estimators trade-off, and closes with a direct side-by-side comparison of bagging and boosting on the Wisconsin Breast Cancer dataset.

File Structure

Each notebook lives in its own folder. The structure within each folder follows a consistent pattern:

Notebook_N/
├── Notebook_N.ipynb          ← the main notebook
├── Challenge_N.ipynb         ← optional exercises (same number as the notebook)
├── visualisations/
│   ├── Figure_1.py           ← interactive figure scripts called from the notebook
│   ├── Figure_2.py
│   └── ...
└── data/
    └── *.csv                 ← datasets used in the notebook

Alongside the notebook folders sit this page (index.html) and the web/ folder, which contains the searchable notebook index, challenge reference, datasets reference, and the how-to guide.

/ (container root)
├── index.html
├── web/
│   ├── search.html           ← searchable index of all topics across all notebooks
│   ├── challenges.html       ← reference guide for all challenge notebooks
│   ├── data.html             ← reference for all datasets used in the series
│   └── how_to.html           ← guide to using the container
├── Notebook_0/
├── Notebook_1/
├── Notebook_2/
│   ...and so on

Python Environment

The container provides a pre-configured Python environment. All libraries are pre-installed and version-pinned so that every notebook runs exactly as written, without any additional setup.

The environment is isolated inside the Docker container and does not affect any Python installation on your host machine.

Viewing and Editing Notebooks

There are many tools capable of opening .ipynb notebook files — including VS Code, PyCharm, Google Colab, and others. This series is designed and tested with JupyterLab, which is included in the container and runs automatically when the container starts. Using a different tool is possible but not officially supported, and some interactive widgets may not render correctly outside JupyterLab.

For instructions on starting the container and connecting to JupyterLab, see the How to Use the Container guide.
